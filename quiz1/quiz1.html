Vai trò JS, biến, toán tử

Câu 1

JavaScript trong trang web chủ yếu phụ trách gì?

✅ Xử lý tương tác và hành vi

HTML tạo cấu trúc, CSS lo giao diện/màu sắc, còn JavaScript xử lý logic, sự kiện và tương tác của người dùng.

Câu 2

Từ khóa nào nên dùng cho giá trị không muốn gán lại?

✅ const

const khai báo hằng — không thể gán lại sau khi khởi tạo. Dùng let khi cần thay đổi, tránh var vì phạm vi dễ gây lỗi.

Câu 3

Toán tử nào nên ưu tiên để so sánh nghiêm ngặt?

✅ ===

=== so sánh cả giá trị lẫn kiểu dữ liệu. == tự ép kiểu dễ sai, còn = là toán tử gán chứ không phải so sánh.
